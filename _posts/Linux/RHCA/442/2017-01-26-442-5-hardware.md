### getconf

用来获取系统的基本配置信息

```
# getconf -a            -> 输出全部系统配置变量值
# getconf PAGE_SIZE     -> 获取到PAGE SIZE大小
4096
# getconf INT_MAX
2147483647
# getconf INT_MIN
-2147483648
# getconf LONG_BIT

# getconf NAME_MAX /usr
255
# getconf PATH_MAX /usr
4096

```
